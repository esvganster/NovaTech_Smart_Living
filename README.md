# NovaTech Smart Living

Landing page en español inspirada en el manual de marca `NovaTech_Smart_Living_Manual_de_Marca.pptx`.

## Contenido

- `index.html`: estructura principal del landing.
- `css/styles.css`: sistema visual futurista basado en la paleta NovaTech.
- `js/main.js`: menú responsive, scroll state y animaciones de entrada.
- `assets/`: logotipos y visuales extraídos del manual de marca.

## Concepto

El sitio traduce los fundamentos del PPTX a una experiencia web:

- propósito: hacer que la inteligencia artificial se sienta natural dentro del hogar;
- promesa: simplificar rutinas, proteger espacios y elevar el bienestar;
- posicionamiento: tecnología premium, accesible y diseñada para la vida real.

## Uso

Abre `index.html` en el navegador para visualizar el proyecto.
